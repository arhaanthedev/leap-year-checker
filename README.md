🚀 Features
Implements the official leap‑year rules:

Divisible by 4 → leap year

Divisible by 100 → not a leap year

Divisible by 400 → leap year

Uses clear conditional logic

Beginner‑friendly structure

Returns human‑readable messages

Easy to extend or integrate into other projects

🧠 How It Works
A year is considered a leap year if:

It is divisible by 4

Unless it is divisible by 100

Unless it is also divisible by 400

This ensures correct handling of special cases like:

2000 → leap year

1900 → not a leap year

2024 → leap year
